## Installation for developers

``` bash
git clone https://github.com/BBG-Deep-Inc/chatgpt.git
```

``` bash
cd chatgpt
python3 -m venv .venv   
source chatgpt/.venv/bin/activate 
```

> [!INFO] 
> Frontend installation

``` bash
cd frontend
pip3 install requirements.txt

```
